# Smart Campus Utility Application

Full-stack campus utility platform built with React, Node.js + Express, PostgreSQL, and JWT authentication.

## Structure

- `backend/` - Express API server
- `frontend/` - React application

## Getting Started

1. Create a PostgreSQL database and set environment variables in `backend/.env`.
2. Run migrations in `backend/sql/init.sql`.
3. Install backend dependencies and start server.
4. Install frontend dependencies and start React app.
